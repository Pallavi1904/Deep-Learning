# 🧠 Deep Learning Che

Welcome to the **Deep Learning Cheatsheets Repository** –  
a well-organized collection of quick-reference guides for **deep learning concepts, architectures, algorithms, and best practices**.

Whether you’re a **student revising for exams**, a **developer working on a project**, or a **researcher who needs a refresher**,  
these cheatsheets are designed to be **concise, visually appealing, and easy to navigate**.

---

## 📌 Purpose

Deep Learning is vast, covering everything from **basic perceptrons** to **transformer-based large language models**.  
The goal of this repository is to:

- Provide **clear and compact reference materials**  
- Help **reduce learning curves** for beginners  
- Act as a **go-to refresher** for experienced practitioners  
- Offer **ready-to-print PDFs and high-quality PNGs** for quick study

---
